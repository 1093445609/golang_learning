准备环境(window):
 1.官网下载go包 msi 安装环境变量一般会自动配置,没有就自己配置变量path  go/bin
    cmd go env 测试环境
 2.因为我是Java学这个所以工具为idea, plugs搜索插件go 有个小熊的图标安装它
 
 3.重启,可以new go文件就行了
 
 4.配置 settings->language->go->goroot
   选择你安装的go路径

  还有个gopath暂时没配置.
